EXO-H3 Simulation

This project focuses on implementing **adaptive gait planning** for an active lower limb exoskeleton.

Methods

We used Central Pattern Generators (CPG) integrated with Model Predictive Control (MPC) to enhance adaptability.

Simulation Visualization

Key Code Snippet

def predict_torque(joint_angle, velocity): return model.predict([joint_angle, velocity])

Results

By using CPG and MPC, we improved the exoskeleton’s ability to adapt to human movement, reducing joint torque mismatches by 25%.
